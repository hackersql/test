Insecure eval function usage

  • The "c" parameter is evaluated with PHP (eval.php?c=sleep(3);)
  • The "c" parameter is evaluated with PHP (blind) (eval-blind.php?c=sleep(3);)
  • The "c" parameter is evaluated with Perl (eval.pl?c=sleep(3);)
  • The "c" parameter is evaluated with Python (eval.py?c=sleep(3);)
