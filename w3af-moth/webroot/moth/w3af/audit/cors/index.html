CORS tests

Please keep in mind that you'll have to send specially crafted HTTP requests in order to use these tests. Your browser won't be able to send them!
  • Test script sending always the same "Access-Control-Allow-Origin" response header value with HTTP 200 response code. (cors_origin_1.php)
  • Test script sending always the same "Access-Control-Allow-Origin" response header value with HTTP 403 response code. (cors_origin_2.php)
  • Test script sending different "Access-Control-Allow-Origin" response header (based on "Origin" request header) value with HTTP 200 response code. (cors_origin_3.php)
  • Test script sending different "Access-Control-Allow-Origin" response header (based on "Origin" request header) value with HTTP 403 response code. (cors_origin_4.php)
