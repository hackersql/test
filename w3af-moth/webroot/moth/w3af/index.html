This file contains links to all test scripts and files used by the w3af framework to perform unit tests. If you find any problem with these scripts, or you discover that w3af is not able to detect one type of XSS, Blind SQL injection, etc. please report a bug here.

Audit plugins

Audit plugins find the vulnerabilities in the web application, examples of vulnerabilities found by this type of plugins are SQL injections and Cross site scripting. Click here to go to the audit plugin test section.

Grep plugins

Grep plugins analyze requests and responses performed by other plugins, and try to find interesting information on them. Click here to go to the grep plugin test section.

Tests related to bruteforce plugins

Bruteforce plugins identify login forms, and protected resources (HTTP 401) in the web application and use a dictionary to try to obtain a valid username and password. Click here to go to the bruteforce plugin test section.

Mangle plugins

Mangle plugins modify the request and response on the fly. Click here to go to the mangle plugin test section.
